# 项目 Agent 工作指令（AgentRules.md）

本文档定义本项目当前 AI 协作 / 测试 / 回归 的固定流程，所有规则均为强约束，除非明确说明，否则不得弱化执行。

## 1、基础原则（Ground Rules）

- 所有改动尽量保持功能更完整、可回滚、可验证。

- 如果使用Unity来开发，凡是需要重复实例化或多处复用的对象（包含UI与非UI），必须创建成prefab，或以可复用模块形式实现。

- 回复、说明、注释等，尽可能使用中文，除非是更利用项目运行的专业化术语、单词等。

- 我是初学者，0基础程序员，请保持代码简洁易懂，并解释关键的保存/读取逻辑。

- 如果由于权限问题，导致你无法执行某些操作，请用详细的步骤指导我如何手动处理。

## 2、总约束声明（重要）

- 本文档中的“必须/不得/不能/不可以”均为硬约束。
